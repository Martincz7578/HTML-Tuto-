Typy elementů

Značky jsou párové nebo nepárové.

Příklady právových:

  • HTML
  • Header
  • Main
  • Footer

Příklady nepárových:

  • Meta
  • Hr
  • Br

In-line elementy

Jde o značky, keteré se používají uvnitř rádků a nevytvářejí nové řádky.

Příklady:

  • tučné (b)
  • kurzíva - zvláštnost (i)
  • zvýraznění (mark)
  • významný pojem (strong)
  • důležitý pojem (em)
  • H2O (sub)
  • 28 (sup)
  • malý (small)
  • HTML (abbr)
  • text pozpátku (bdo)

Blokové elementy

Příklady:

  • Seznamy
  • p
  • Nadpisy
